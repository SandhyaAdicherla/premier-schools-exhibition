# School Exhibition Landing Page

## Overview

This project is a responsive landing page developed as part of the Frontend Developer assignment.

## Features

* Responsive design for desktop, tablet, and mobile devices
* Horizontal card carousel with navigation controls
* Smooth scrolling interactions
* Mobile-optimized layouts
* Semantic HTML and accessible UI elements

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## Project Setup

1. Clone the repository:
   git clone <repository-url>

2. Open the project folder.

3. Run using a local server (e.g., VS Code Live Server).
